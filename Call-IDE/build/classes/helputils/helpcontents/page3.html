Building Files

Building is easy and simple in Call-IDE.

Compilation: Things to remember.

  • You shouldn't compile before saving your current files
  • You shouldn't run your project before compiling the current files
  • The IDE will be reminding you for these cases

Build Tasks

  • Your build files will be stored in your project's build file
  • You can create jar files with dependencies compressed into one single jar file in your project's dist folder.
  • You can create javadoc of your project, it will be saved in your projects doc folder

Customizing Builds

  • Call-IDE uses Ant-Scripts to compile your projects , you can access your build xml files in your Call-IDE folder located in your system home folder
