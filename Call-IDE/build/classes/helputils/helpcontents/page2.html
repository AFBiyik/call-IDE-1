Editing Text

You can easily use the common functionalities (e.g. cut, copy, paste, undo, redo, find & replace) by using the hotkeys, the Edit menu or the toolbar.
The editor's font can be changed by choosing Tools > Preferences.
There are also various theme options with different coloring in Preferences.

Generating Code Templates

Code Templates and Insert Javadoc Comment operations under the Tools menu can be used to insert pre-determined text snippets to the current document.

Auto-complete Functionality

Whenever you save a Java source code to a directory, Call-IDE checks for the properties of the saved class and enables Auto-complete functionality on the editor.
This auto-complete popup menu can be accessed by pressing CTRL+SPACE while editing.
