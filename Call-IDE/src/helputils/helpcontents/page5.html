Managing Projects


How to Create Java Projects

A project is a set of source files and adjustments with which you can build and run them.
Call-IDE uses an automatically generated ANT script to handle building operations. (For more information: http://ant.apache.org/)
You can create a new project by simply choosing File > New Project and selecting the project location & name.

Generating Jar File

An executable jar file is created by choosing Tools > Generate Jar File or pressing generate jar file button on the toolbar.

Generating Javadoc Comment

Javadoc is a tool for generating API documentation in HTML format from the javadoc comments in source code. To generate javadoc HTML files from the source code, choose Tools > Generate Javadoc or press generate javadoc button on the toolbar.
